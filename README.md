<h3>Вызов плагина</h3>
```javascript
$('.class').dragger();
```

<h3>Настройки</h3>
```javascript
$('.dragger1').dragger({
	starClass: 'dragger-start1',
	endClass: 'dragger-end1',
	dragClass: 'drag',
	drag: true
});
```

<h3>Html</h3>
```html
<div class="dragger">
	<img src="dragger-start.png">
	<img src="dragger-end.png">
</div>
```
