## Вызов плагина
```javascript
$('.class').dragger();
```

## Настройки
```javascript
$('.dragger').dragger({
	starClass: 'dragger-start',
	endClass: 'dragger-end',
	dragClass: 'drag',
	drag: true
});
```

## Html
```html
<div class="dragger">
	<img src="dragger-start.png">
	<img src="dragger-end.png">
</div>
```
